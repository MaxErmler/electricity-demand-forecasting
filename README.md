# electricity-demand-forecasting
Benchmarked 10+ models — OLS, Ridge, LASSO, PCR, WLS, Huber, LAD, LTS, SARIMA, SARIMAX, and a PyTorch LSTM — against 50,000+ hours of Toledo Edison utility data. LSTM achieved 78,432 KW RMSE, 77% better than baseline, ~$115M/yr in reduced market exposure. R and Python
